自分用のMIDIビジュアライザーなどの詰め合わせです。

自分以外が使うことを想定して作ってないのでデプロイなんてしてません。自分で環境組んでください

面倒なので自分でプログラム読んでimportで書いてある奴をインストールしてください。多分Python3系なら動きます。


困ったらGeminiに聞いてください、たぶん動くので困ることないと思いますが

使い道としては曲のMVやピアノの練習動画などを想定してます。

各種パラメータは結構ハードコードされてるので好みに合わせて直接プログラムを編集してください。見ればわかります。

出力される動画は仕様上透過できなかったのでご自身の編集ソフトのクロマキー機能を使ってください。背景色自体は変更可能なので自分でBBやGBにしてください。

説明
- midi_visualizer.py
  
  シンプルな鍵盤のビジュアライザーです。MIDIと出力先のフォルダを指定すれば動きます。

  実装はOpenCVとPyGameです。
  
- midi_visualizer_moviepy.py
  
  上のものに加えて、黒鍵と白鍵の代わりに任意の画像や動画を読み込めます。透過のものはそのまま透過して重ねて表示されます。
  
  位置も自由に調整できるので表現の幅は意外と大きいです。これの出力を複数重ねてもいいかも
  
  実装方法がmoviePyなのでこの名前です。
